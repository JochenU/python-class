Most Python programmers are familiar with lists, dictionaries, tuples, and other basic datatypes. In this exercise, we briefly look at some of the other container types in the standard library.

(a) Counter objects and tabulation

Counter objects are useful if you want to tabulate things. To illustrate, the file Data/words.txt contains the following text:

look into my eyes
look into my eyes
the eyes the eyes the eyes
not around the eyes
don't look around the eyes
look into my eyes you're under

Use a Counter to quickly tabulate the word counts:

>>> words = open('Data/word.txt').read().split()
>>> wc = Counter(words)
>>> wc['eyes']
8
>>> wc['look']
4
>>> wc.most_common()
[('eyes', 8), ('the', 5), ('look', 4), ('into', 3), ('my', 3), ('around', 2), ("you're", 1), ("don't", 1), ('under', 1), ('not', 1)]
>>>

(b) Using defaultdicts to build an index

defaultdict objects are useful whenever you need to store multiple values for a dictionary key. This frequently arises when building indices. For example, let’s build an index that gives us the positions of each word in the previous example:

>>> from collections import defaultdict
>>> windex = defaultdict(list)
>>> n = 0
>>> for word in words:
        windex[word].append(n)
        n += 1

>>> windex['eyes']
[3, 7, 9, 11, 13, 17, 22, 26]
>>> windex['look']
[0, 4, 19, 23]
>>> windex['around']
[15, 20]
>>> words[15]
'around'
>>> words[20]
'around'
>>>

(d) Mondo counters/defaultdicts

Counters and defaultdict objects can be combined together to very powerful effect. Consider the following code which tabulates bus ridership data by year and then by bus route:

import readrides
from collections import defaultdict, Counter

rides = readrides.read_rides_as_dicts('Data/ctabus.csv')

yearly_ridership = defaultdict(Counter)
for ride in rides:
    month, day, year = ride['date'].split('/')
    yearly_ridership[year][ride['route']] += ride['numrides']

This code builds a data structure similar to that in part (c) of Exercise 2.2:

>>> yearly_ridership['2011']['22']
7392947
>>> yearly_ridership['2001']['22']
6862942
>>> yearly_ridership['2011']['22'] - yearly_ridership['2001']['22']
530005
>>>

However, the use of Counters makes it even easier to perform more complex operations. For example, here are the five bus routes that experienced the greatest increase in ridership from 2001-2011.

>>> (yearly_ridership['2011'] - yearly_ridership['2001']).most_common(5)
[('15', 2732209), ('147', 2107910), ('66', 1612958), ('12', 1612067), ('14', 1351308)]
>>>

Marvel at how a rather complex operation was expressed in a single line of code.

Discussion

There is much more to the collections module than what has been presented here (e.g., deque, OrderedDict, etc.). When faced with a tricky data-structures problem, collections is always worth a look. You may find that your problem has already been solved.
